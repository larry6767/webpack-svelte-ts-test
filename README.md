# Webpack Svelte TypeScript Test

## Recommended working environment

```bash
node -v # 18.16.0
yarn -v # 1.22.19
```

Don't forget to create `.env` file by [.env.example](./.env.example)!

## Usage

1. Start project:

   ```bash
    yarn start
   ```

2. Build project:

   ```bash
    yarn build
   ```
